انشر OpenClaw على Northflank باستخدام قالب بنقرة واحدة وادخل إليه عبر الويب باستخدام Control UI.
هذا هو أسهل مسار "من دون طرفية على الخادم": يقوم Northflank بتشغيل Gateway نيابةً عنك.

## كيفية البدء

1. انقر على [Deploy OpenClaw](https://northflank.com/stacks/deploy-openclaw) لفتح القالب.
2. أنشئ [حسابًا على Northflank](https://app.northflank.com/signup) إذا لم يكن لديك واحد بالفعل.
3. انقر على **Deploy OpenClaw now**.
4. اضبط متغير البيئة المطلوب: `OPENCLAW_GATEWAY_TOKEN` ‏(استخدم قيمة عشوائية قوية).
5. انقر على **Deploy stack** لبناء قالب OpenClaw وتشغيله.
6. انتظر حتى يكتمل النشر، ثم انقر على **View resources**.
7. افتح خدمة OpenClaw.
8. افتح عنوان URL العام لـ OpenClaw عند `/openclaw` واتصل باستخدام السر المشترك المهيأ. يستخدم هذا القالب `OPENCLAW_GATEWAY_TOKEN` افتراضيًا؛ وإذا استبدلته بمصادقة كلمة المرور، فاستخدم كلمة المرور تلك بدلًا منه.

## ما الذي ستحصل عليه

- Gateway + Control UI مستضافان لـ OpenClaw
- تخزين دائم عبر Northflank Volume ‏(`/data`) بحيث تبقى `openclaw.json`،
  و`auth-profiles.json` لكل وكيل، وحالة القنوات/المزوّدين، والجلسات،
  ومساحة العمل محفوظة عبر عمليات إعادة النشر

## توصيل قناة
