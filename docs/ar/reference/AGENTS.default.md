# AGENTS.md - مساعد OpenClaw الشخصي (الافتراضي)

## التشغيل الأول (مستحسن)

يستخدم OpenClaw دليل مساحة عمل مخصصًا للوكيل. الافتراضي: `~/.openclaw/workspace` (يمكن ضبطه عبر `agents.defaults.workspace`).

1. أنشئ مساحة العمل (إذا لم تكن موجودة بالفعل):

```bash
mkdir -p ~/.openclaw/workspace
```

2. انسخ قوالب مساحة العمل الافتراضية إلى مساحة العمل:

```bash
cp docs/reference/templates/AGENTS.md ~/.openclaw/workspace/AGENTS.md
cp docs/reference/templates/SOUL.md ~/.openclaw/workspace/SOUL.md
cp docs/reference/templates/TOOLS.md ~/.openclaw/workspace/TOOLS.md
```

3. اختياري: إذا كنت تريد قائمة Skills الخاصة بالمساعد الشخصي، فاستبدل AGENTS.md بهذا الملف:

```bash
cp docs/reference/AGENTS.default.md ~/.openclaw/workspace/AGENTS.md
```

4. اختياري: اختر مساحة عمل مختلفة عبر ضبط `agents.defaults.workspace` (يدعم `~`):

```json5
{
  agents: { defaults: { workspace: "~/.openclaw/workspace" } },
}
```

## ما الذي يفعله OpenClaw

- يشغّل بوابة WhatsApp + وكيل Pi البرمجي حتى يتمكن المساعد من قراءة/كتابة الدردشات، وجلب السياق، وتشغيل Skills عبر مضيف Mac.
- يدير تطبيق macOS الأذونات (تسجيل الشاشة، والإشعارات، والميكروفون) ويكشف CLI الأمر `openclaw` عبر الملف التنفيذي المضمّن فيه.
- تُدمج الدردشات المباشرة في جلسة `main` الخاصة بالوكيل افتراضيًا؛ وتبقى المجموعات معزولة على شكل `agent:<agentId>:<channel>:group:<id>` (الغرف/القنوات: `agent:<agentId>:<channel>:channel:<id>`); وتحافظ heartbeats على المهام الخلفية حيّة.

## Skills الأساسية (فعّلها في Settings → Skills)

- **mcporter** — وقت تشغيل/CLI لخادم الأدوات من أجل إدارة خلفيات Skills الخارجية.
- **Peekaboo** — لقطات شاشة سريعة لـ macOS مع تحليل اختياري للرؤية بالذكاء الاصطناعي.
- **camsnap** — التقاط إطارات أو مقاطع أو تنبيهات حركة من كاميرات الأمان RTSP/ONVIF.
- **oracle** — CLI لوكيل جاهز لـ OpenAI مع إعادة تشغيل الجلسة والتحكم في المتصفح.
- **eightctl** — تحكم في نومك، من الطرفية.
- **imsg** — إرسال iMessage وSMS وقراءتهما وبثهما.
- **wacli** — CLI لـ WhatsApp: مزامنة، بحث، إرسال.
- **discord** — إجراءات Discord: تفاعلات، وملصقات، واستطلاعات. استخدم أهداف `user:<id>` أو `channel:<id>` (المعرّفات الرقمية المجردة ملتبسة).
- **gog** — CLI لـ Google Suite: Gmail وCalendar وDrive وContacts.
- **spotify-player** — عميل Spotify للطرفية للبحث/الطوابير/التحكم في التشغيل.
- **sag** — كلام ElevenLabs مع تجربة استخدام شبيهة بـ say على mac؛ ويقوم بالبث إلى السماعات افتراضيًا.
- **Sonos CLI** — التحكم في سماعات Sonos (الاكتشاف/الحالة/التشغيل/الصوت/التجميع) من النصوص البرمجية.
- **blucli** — تشغيل وتجميع وأتمتة مشغلات BluOS من النصوص البرمجية.
- **OpenHue CLI** — التحكم في إضاءة Philips Hue للمشاهد وعمليات الأتمتة.
- **OpenAI Whisper** — تحويل الكلام إلى نص محليًا للإملاء السريع ونصوص البريد الصوتي.
- **Gemini CLI** — نماذج Google Gemini من الطرفية من أجل أسئلة وأجوبة سريعة.
- **agent-tools** — مجموعة أدوات مساعدة للأتمتة والنصوص البرمجية المساعدة.

## ملاحظات الاستخدام

- فضّل CLI الأمر `openclaw` للبرمجة النصية؛ إذ يتولى تطبيق mac الأذونات.
- شغّل التثبيتات من تبويب Skills؛ فهو يخفي الزر إذا كان الملف التنفيذي موجودًا بالفعل.
- أبقِ heartbeats مفعّلة حتى يتمكن المساعد من جدولة التذكيرات، ومراقبة صناديق الوارد، وتشغيل التقاطات الكاميرا.
- تعمل واجهة Canvas بملء الشاشة مع طبقات أصلية. تجنب وضع عناصر التحكم الحرجة في الحواف العلوية اليسرى/اليمنى أو السفلية؛ أضف هوامش صريحة في التخطيط ولا تعتمد على safe-area insets.
- بالنسبة إلى التحقق المعتمد على المتصفح، استخدم `openclaw browser` ‏(علامات تبويب/حالة/لقطة شاشة) مع ملف تعريف Chrome الذي يديره OpenClaw.
- بالنسبة إلى فحص DOM، استخدم `openclaw browser eval|query|dom|snapshot` ‏(واستخدم `--json`/`--out` عندما تحتاج إلى خرج آلي).
- بالنسبة إلى التفاعلات، استخدم `openclaw browser click|type|hover|drag|select|upload|press|wait|navigate|back|evaluate|run` ‏(يتطلب click/type مراجع snapshot؛ واستخدم `evaluate` لمحددات CSS).
