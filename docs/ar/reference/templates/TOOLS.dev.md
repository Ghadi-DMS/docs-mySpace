# TOOLS.md - ملاحظات أدوات المستخدم (قابل للتعديل)

هذا الملف مخصص لملاحظاتك _أنت_ حول الأدوات والاصطلاحات الخارجية.
وهو لا يحدد الأدوات الموجودة؛ إذ يوفر OpenClaw الأدوات المضمنة داخليًا.

## أمثلة

### imsg

- إرسال iMessage/SMS: صف إلى من/ماذا، وأكّد قبل الإرسال.
- فضّل الرسائل القصيرة؛ وتجنب إرسال الأسرار.

### sag

- تحويل النص إلى كلام: حدّد الصوت، ومكبر الصوت/الغرفة المستهدفة، وما إذا كان يجب البث.

أضف أي شيء آخر تريد أن يعرفه المساعد عن سلسلة الأدوات المحلية لديك.
