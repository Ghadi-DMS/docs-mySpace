# TOOLS.md - ملاحظات محلية

تحدد Skills _كيفية_ عمل الأدوات. هذا الملف مخصص لتفاصيلك _أنت_ — الأشياء الفريدة في إعدادك.

## ما الذي يوضع هنا

أشياء مثل:

- أسماء الكاميرات ومواقعها
- مضيفات SSH وأسماؤها المستعارة
- الأصوات المفضلة لتحويل النص إلى كلام
- أسماء السماعات/الغرف
- الأسماء المستعارة للأجهزة
- أي شيء خاص بالبيئة

## أمثلة

```markdown
### الكاميرات

- living-room → المنطقة الرئيسية، زاوية عريضة 180°
- front-door → المدخل، يُفعّل بالحركة

### SSH

- home-server → 192.168.1.100، المستخدم: admin

### TTS

- الصوت المفضل: "Nova" (دافئ، بلمسة بريطانية خفيفة)
- السماعة الافتراضية: Kitchen HomePod
```

## لماذا الفصل؟

Skills مشتركة. أما إعدادك فهو لك. إن فصلها عن بعضها يعني أنه يمكنك تحديث Skills دون فقدان ملاحظاتك، ومشاركة Skills دون كشف بنيتك التحتية.
