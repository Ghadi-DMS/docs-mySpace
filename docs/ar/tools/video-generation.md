# توليد الفيديو

يمكن لوكلاء OpenClaw إنشاء مقاطع فيديو من مطالبات نصية، أو صور مرجعية، أو مقاطع فيديو موجودة. يتم دعم اثنتي عشرة واجهة مزودين خلفية، ولكل منها خيارات نماذج وأنماط إدخال ومجموعات ميزات مختلفة. يختار الوكيل المزود المناسب تلقائيًا بناءً على إعداداتك ومفاتيح API المتاحة.

<Note>
لا تظهر أداة `video_generate` إلا عند توفر مزود واحد على الأقل لتوليد الفيديو. إذا لم ترها ضمن أدوات الوكيل لديك، فاضبط مفتاح API لمزود أو هيئ `agents.defaults.videoGenerationModel`.
</Note>

## بدء سريع

1. اضبط مفتاح API لأي مزود مدعوم:

```bash
export GEMINI_API_KEY="your-key"
```

2. اختياريًا، ثبّت نموذجًا افتراضيًا:

```bash
openclaw config set agents.defaults.videoGenerationModel.primary "google/veo-3.1-fast-generate-preview"
```

3. اطلب من الوكيل:

> أنشئ فيديو سينمائيًا مدته 5 ثوانٍ لكركند ودود يتزلج على الأمواج وقت الغروب.

سيستدعي الوكيل `video_generate` تلقائيًا. لا حاجة إلى قائمة سماح للأدوات.

## ما الذي يحدث عند إنشاء فيديو

توليد الفيديو غير متزامن. عندما يستدعي الوكيل `video_generate` داخل جلسة:

1. يرسل OpenClaw الطلب إلى المزود ويعيد فورًا معرّف مهمة.
2. يعالج المزود المهمة في الخلفية (عادةً من 30 ثانية إلى 5 دقائق بحسب المزود والدقة).
3. عندما يصبح الفيديو جاهزًا، يوقظ OpenClaw الجلسة نفسها بحدث إكمال داخلي.
4. ينشر الوكيل الفيديو النهائي مرة أخرى في المحادثة الأصلية.

أثناء وجود مهمة قيد التنفيذ، تعيد استدعاءات `video_generate` المكررة داخل الجلسة نفسها حالة المهمة الحالية بدلًا من بدء توليد آخر. استخدم `openclaw tasks list` أو `openclaw tasks show <taskId>` للتحقق من التقدم عبر CLI.

خارج تشغيلات الوكيل المدعومة بجلسة (على سبيل المثال، الاستدعاءات المباشرة للأدوات)، تعود الأداة إلى التوليد المضمن وتعيد مسار الوسائط النهائي في الدور نفسه.

## معلمات الأداة

### مطلوبة

| المعلمة | النوع | الوصف |
| --------- | ------ | ----------------------------------------------------------------------------- |
| `prompt`  | string | وصف نصي للفيديو المطلوب إنشاؤه (مطلوب عند `action: "generate"`) |

### مدخلات المحتوى

| المعلمة | النوع | الوصف |
| --------- | -------- | ------------------------------------ |
| `image`   | string   | صورة مرجعية واحدة (مسار أو URL) |
| `images`  | string[] | صور مرجعية متعددة (حتى 5) |
| `video`   | string   | فيديو مرجعي واحد (مسار أو URL) |
| `videos`  | string[] | مقاطع فيديو مرجعية متعددة (حتى 4) |

### عناصر التحكم في النمط

| المعلمة | النوع | الوصف |
| ----------------- | ------- | ------------------------------------------------------------------------ |
| `aspectRatio`     | string  | `1:1`, `2:3`, `3:2`, `3:4`, `4:3`, `4:5`, `5:4`, `9:16`, `16:9`, `21:9` |
| `resolution`      | string  | `480P` أو `720P` أو `1080P` |
| `durationSeconds` | number  | المدة المستهدفة بالثواني (تُقرَّب إلى أقرب قيمة مدعومة من المزود) |
| `size`            | string  | تلميح حجم عندما يدعمه المزود |
| `audio`           | boolean | تفعيل الصوت المولّد عندما يكون مدعومًا |
| `watermark`       | boolean | تبديل العلامة المائية الخاصة بالمزود عندما يكون ذلك مدعومًا |

### متقدمة

| المعلمة | النوع | الوصف |
| ---------- | ------ | ----------------------------------------------- |
| `action`   | string | `"generate"` (الافتراضي) أو `"status"` أو `"list"` |
| `model`    | string | تجاوز المزود/النموذج (مثل `runway/gen4.5`) |
| `filename` | string | تلميح اسم ملف الإخراج |

ليست كل المعلمات مدعومة لدى جميع المزودين. يتم تجاهل التجاوزات غير المدعومة على أساس أفضل جهد ويُبلّغ عنها كتحذيرات في نتيجة الأداة. أما حدود القدرات الصارمة (مثل وجود عدد كبير جدًا من المدخلات المرجعية) فتفشل قبل الإرسال.

## الإجراءات

- **generate** (الافتراضي) -- أنشئ فيديو من prompt المعطى والمدخلات المرجعية الاختيارية.
- **status** -- تحقّق من حالة مهمة الفيديو قيد التنفيذ للجلسة الحالية من دون بدء توليد آخر.
- **list** -- اعرض المزودين والنماذج وقدراتهم المتاحة.

## اختيار النموذج

عند إنشاء فيديو، يحل OpenClaw النموذج بهذا الترتيب:

1. **معلمة الأداة `model`** -- إذا حدد الوكيل واحدة في الاستدعاء.
2. **`videoGenerationModel.primary`** -- من الإعدادات.
3. **`videoGenerationModel.fallbacks`** -- تُجرَّب بالترتيب.
4. **الاكتشاف التلقائي** -- يستخدم المزودين الذين يملكون مصادقة صالحة، بدءًا بالمزود الافتراضي الحالي، ثم المزودين المتبقين بترتيب أبجدي.

إذا فشل أحد المزودين، تُجرَّب المحاولة التالية تلقائيًا. وإذا فشل جميع المرشحين، يتضمن الخطأ تفاصيل من كل محاولة.

```json5
{
  agents: {
    defaults: {
      videoGenerationModel: {
        primary: "google/veo-3.1-fast-generate-preview",
        fallbacks: ["runway/gen4.5", "qwen/wan2.6-t2v"],
      },
    },
  },
}
```

## ملاحظات حول المزودين

| المزود | ملاحظات |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| Alibaba  | يستخدم نقطة النهاية غير المتزامنة DashScope/Model Studio. يجب أن تكون الصور ومقاطع الفيديو المرجعية عناوين URL بعيدة من نوع `http(s)`. |
| BytePlus | يدعم مرجع صورة واحدة فقط. |
| ComfyUI  | تنفيذ محلي أو سحابي معتمد على workflow. يدعم التحويل من نص إلى فيديو ومن صورة إلى فيديو عبر الرسم البياني المُعد. |
| fal      | يستخدم تدفقًا معتمدًا على قائمة الانتظار للمهام طويلة التشغيل. يدعم مرجع صورة واحدة فقط. |
| Google   | يستخدم Gemini/Veo. يدعم صورة مرجعية واحدة أو فيديو مرجعي واحد. |
| MiniMax  | يدعم مرجع صورة واحدة فقط. |
| OpenAI   | يتم تمرير تجاوز `size` فقط. أما تجاوزات النمط الأخرى (`aspectRatio`, `resolution`, `audio`, `watermark`) فيتم تجاهلها مع تحذير. |
| Qwen     | يستخدم الواجهة الخلفية DashScope نفسها مثل Alibaba. يجب أن تكون المدخلات المرجعية عناوين URL بعيدة من نوع `http(s)`؛ ويتم رفض الملفات المحلية مسبقًا. |
| Runway   | يدعم الملفات المحلية عبر data URIs. يتطلب التحويل من فيديو إلى فيديو `runway/gen4_aleph`. تعرض التشغيلات النصية فقط نسب العرض إلى الارتفاع `16:9` و`9:16`. |
| Together | يدعم مرجع صورة واحدة فقط. |
| Vydra    | يستخدم `https://www.vydra.ai/api/v1` مباشرة لتجنب عمليات إعادة التوجيه التي تُسقط المصادقة. يتم تضمين `veo3` كتحويل من نص إلى فيديو فقط؛ ويتطلب `kling` عنوان URL بعيدًا لصورة. |
| xAI      | يدعم التحويل من نص إلى فيديو، ومن صورة إلى فيديو، وتدفقات تحرير/تمديد الفيديو البعيد. |

## الإعدادات

اضبط نموذج توليد الفيديو الافتراضي في إعدادات OpenClaw:

```json5
{
  agents: {
    defaults: {
      videoGenerationModel: {
        primary: "qwen/wan2.6-t2v",
        fallbacks: ["qwen/wan2.6-r2v-flash"],
      },
    },
  },
}
```

أو عبر CLI:

```bash
openclaw config set agents.defaults.videoGenerationModel.primary "qwen/wan2.6-t2v"
```
